CPU Scheduling Simulator
Developed a Java-based desktop application to simulate and visualize Operating System process scheduling algorithms, including Round Robin (RR) and Shortest Remaining Time First (SRTF).
Implemented a dynamic Gantt Chart visualization to represent execution sequences and context switching in real-time.
Integrated a comparison engine to analyze performance metrics such as Average Waiting Time (WT), Turnaround Time (TAT), and Response Time (RT).
Built an interactive GUI allowing users to input process parameters (Arrival & Burst time) and compare algorithm efficiency through automated summary reports.
